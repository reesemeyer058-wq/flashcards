Flashcards (3×5)

Print

Print steps

  1. Click Print fronts.
  2. Put the same stack back into your printer.
  3. Click Print backs.
  • Scale: 100% (turn OFF “Fit to page”).
  • Prints on Letter paper as 4 outlined 3×5 cut boxes.
  • Text is rotated so it reads like a traditional flashcard.

Manual duplex options

Use Print test sheet first. If pairing is wrong, change Back order. If upside-down, change Flip direction.
